Elementos HTML en línea

HTML define una larga lista de etiquetas en línea disponibles, una lista completa de las cuales se puede encontrar en Mozilla Developer Network .

  • Para texto en negrita , use <strong> .
  • Para poner el texto en cursiva , utilice <em> .
  • Se deben utilizar abreviaturas, como HTML <abbr> , con un title atributo opcional para la frase completa.
  • Citas, como - Mark Otto , deben usarse <cite> .
  • Eliminado el texto debe usar <del> y insertado el texto debe usar <ins> .
  • Usos de texto en superíndice <sup> y usos de texto en subíndice <sub> .
